### 快速推断天干地支年

- 快速推算年柱干支法
- 公元后任何年代：公元年末位数-3=年干，负数则加10
- 公元前任何年代：公元年末位数+8=年干 
- 推算地支：
- 公元后任何年代：（公元年数-3）÷12，得余数0-11为年支，负数则加12，0视做12
- 公元前任何年代：（公元年数-2）÷12，余数-11-0为年支，加12转为正数，0视做12
